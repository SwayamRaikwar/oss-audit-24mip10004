## 📌 Project Description
This project is based on the study and analysis of an open-source software (Git). It includes understanding its origin, license, and role in the open-source ecosystem. The project also includes practical implementation using Linux shell scripting.

## 📜 Scripts Included

### 1. Script 1 - System Identity Report
Displays system information like kernel version, user, and uptime.

### 2. Script 2 - Package Inspector
Checks whether Git is installed and shows its version.

### 3. Script 3 - Disk and Permission Auditor
Displays disk usage and permissions of important directories.

### 4. Script 4 - Log File Analyzer
Analyzes log files and counts occurrences of a keyword.

### 5. Script 5 - Manifest Generator
Takes user input and generates a personalized open-source message.
